数据结构之树

周六 04 四月 2015 by lhqqqqq

一、概念

树（tree）：由n（n>=1）个有限节点组成一个具有层次关系的集合；

度：节点的子树个数；

树的度：树中任意节点的度的最大值；

兄弟：两节点的父节点相同；

层：根在第一层，以此类推；

高度：叶子节点的高度为1，根节点高度最高；

二叉树：每个节点最多有两个子树的树

满二叉树：叶子节点一定要在最后一层，并且所有非叶子节点都存在左孩子和右孩子；

完全二叉树：除最后一层外，每一层上的节点数均达到最大值；在最后一层上只缺少右边的若干结点。

二、二叉树性质

  1. 在非空二叉树中，第i层的结点总数不超过2i-1, i>=1；
  2. 深度为h的二叉树最多有2h-1个结点(h>=1)，最少有h个结点；
  3. 对于任意一棵二叉树，如果其叶结点数为N0，而度数为2的结点总数为N2，则N0=N2+1；
  4. 具有n个结点的完全二叉树的深度为log2(n+1)
  5. 有N个结点的完全二叉树各结点如果用顺序方式存储，若I为结点编号，则结点之间有如下关系：
    • 如果I>1，则其父结点的编号为I/2；
    • 如果2*I<=N，则其左儿子（即左子树的根结点）的编号为2*I
    • 若2*I>N，则无左儿子；
    • 如果2*I+1<=N，则其右儿子的结点编号为2*I+1；若2*I+1>N，则无右儿子。
  6. 给定N个节点，能构成h(N)种不同的二叉树。 h(N)为卡特兰数的第N项。h(n)=C(2*n，n)/(n+1)。
  7. 设有i个枝点，I为所有枝点的道路长度总和，J为叶的道路长度总和J=I+2i。

三、实现

树节点实现

struct TreeNode  
{  
    int val;  
    TreeNode *left;  
    TreeNode *right;  
    TreeNode(int val):val(val), left(NULL),right(NULL){} ;   
};

前序遍历（PreorderTraversal），根->左->右

void PreorderTraversal(TreeNode *node)  
{  
    if(!node)  
        return; 
    Visit(T); 
    PreorderTraversal(node->left);  
    PreorderTraversal(node->right);  
}

中序遍历（InorderTraversal），左->根->右

void PreorderTraversal(TreeNode *node)  
{  
    if(!node)  
        return; 
    PreorderTraversal(node->left);  
    Visit(T);       
    PreorderTraversal(node->right);  
}

后序遍历（PostorderTraversal），左->右->根

void PreorderTraversal(TreeNode *node)  
{  
    if(!node)  
        return;
    PreorderTraversal(node->left);  
    PreorderTraversal(node->right); 
    Visit(T);   
}
