C++11新特性记录

 周三 08 四月 2015 by lhqqqqq

  1. C++11

    C++11标准是 ISO/IEC 14882:2011 - Information technology -- Programming languages -- C++ 的简称。 C++11标准由国际标准化组织（ISO）和国际电工委员会（IEC）旗下的C++标准委员会（ISO/IEC JTC1/SC22/WG21）于2011年8月12日公布，并于2011年9月出版。2012年2月28日的国际标准草案(N3376)是最接近于C++11标准的草案（仅编辑上的修正）。此次标准为C++98发布后13年来第一次重大修正。

  2. auto & decltype

    auto：让编译器分析表达式所属的类型。auto定义的变量必须有初始值。

    auto i = 1, *p = &i; //整型，整型指针

    应用在容器中

    auto len = str.size(); //len的类型是string::size_type

    应用在容器的迭代器中

    vector<int> v{1, 2, 3};
for （auto it = v.begin(); it != v.end; ++i）
    (*it)++;
//it的类型为vector<int>::iterator

    decltype：选择并返回操作数的数据类型。

    decltype(f()) sum = x;

    编译器不实际调用函数f，而是使用f返回值类型来作为sum的类型。

    decltype((variable))的结果永远是引用，decltype(variable)的结果只有当variable本身是引用时才是引用。

  3. 范围for语句

    遍历给定序列中的每个元素并对序列中的每个值执行某种操作

    for (declaration: expression)
    statement

    expression部分是一个对象，用于表示一个序列；declaration部分负责定义一个变量，该变量将被用于访问序列中的基础元素。 如：

    string str("Hello world!");
for (auto c : str)
    cout << c <<endl;

  4. 列表初始化

    列表初始化不再局限于数组。无论是初始化对象还是某些时候为新对象赋值，都可以使用一组花括号括起来的初始值。

    如，容器初始化

    vector<int> v = {1, 2, 3};

    或

    vector<int> v{1, 2, 3};

    取代以前的方法：

    int arr[3] = {1, 2, 3}  
vector<int> v(arr, arr + 3);

周二 07 四月 2015 by lhqqqqq

from 《C++ Primer》

一、概念

引用（reference）的符号为&，一般指“左值引用”，作用是为对象起一个别名。

C++11引入了一种“右值引用”

定义引用时，程序把初始值和对象绑定在一次。之后，这种绑定关系会一直存在，所以引用必须初始化。 引用本身不是对象。

二、示例

int ival = 1024;
int &refVal = ival; //引用，即refVal是ival的别名
int &refVal2; //报错，引用必须被初始化
int &refVal3 = 10; //报错，引用的初始值必须是对象
double dval = 3.14;
int &refVal4 = dval; //报错，类型不一致

数据结构之树
周六 04 四月 2015 by lhqqqqq

一、概念

树（tree）：由n（n>=1）个有限节点组成一个具有层次关系的集合；

度：节点的子树个数；

树的度：树中任意节点的度的最大值；

兄弟：两节点的父节点相同；

层：根在第一层，以此类推；

高度：叶子节点的高度为1，根节点高度最高；

二叉树：每个节点最多有两个子树的树

满二叉树：叶子节点一定要在最后一层，并且所有非叶子节点都存在左孩子和右孩子；

完全二叉树：除最后一层外，每一层上的节点数均达到最大值；在最后一层上只缺少右边的若干结点。

二、二叉树性质

周二 24 三月 2015 by lhqqqqq
经典的IPC问题

1965年，Dijkstra提出并解决了一个他称之为哲学家就餐的同步问题。从那时起，每个发明新的同步原语的人都希望通过解决哲学家就餐问题来展示其同步原语的精妙之处。这个问题可以简单地描述如下：五个哲学家围坐在一张圆桌周围，每个哲学家面前都有一盘通心粉。由于通心粉很滑，所以需要两把叉子才能夹住。相邻两个盘子之间放有一把叉子，餐桌如图2-44所示。

哲学家的生活中有两种交替活动时段：即吃饭和思考（这只是一种抽象，即对哲学家而言其他活动都无关紧要）。当一个哲学家觉得饿了时，他就试图分两次去取其左边和右边的叉子，每次拿一把，但不分次序。如果成功地得到了两把叉子，就开始吃饭，吃完后放下叉子继续思考。关键问题是：能为每一个哲学家写一段描述其行为的程序，且决不会死锁吗？（要求拿两把叉子是人为规定的，我们也可以将意大利面条换成中国菜，用米饭代替通心粉，用筷子代替叉子。）
